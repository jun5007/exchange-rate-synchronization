# 환율 민감도 기반 클러스터링과 동조지수를 이용한 산업별 월간 수익률 예측

> Exchange Rate Sensitivity Clustering and Monthly Industry Return Prediction Using Synchronization Index  
> 2025년 2학년 1학기 팀 프로젝트  
> 원본 데이터와 논문 전문은 저작권 및 팀 자료 공개 범위 확인 전까지 포함하지 않습니다.

---

## 프로젝트 개요

- 프로젝트명: 환율 민감도 기반 클러스터링과 동조지수를 이용한 산업별 월간 수익률 예측
- 진행 기간: 2025.03 - 2025.06
- 소속: 순천향대학교 AI빅데이터학과
- 형태: 팀 프로젝트 / 논문 형식 결과물
- 주제:
  - KOSPI 대형주를 환율 민감도와 외국인 수급 기준으로 군집화
  - 산업 간 동조 현상을 Lift와 Pearson 상관계수 기반 `mean_corr`로 정량화
  - 동조지수를 월간 산업 수익률 예측 모델에 반영해 성능 변화를 비교

## 데이터 처리 및 피처 엔지니어링

### 주요 파생 피처

| 구분 | 변수 | 설명 |
|---|---|---|
| 수익률 | `mean_ret` | 산업별 월평균 수익률 |
| 환율 | `mean_fx` | 월평균 USD/KRW 변동률 |
| 수급 | `mean_flow` | 월평균 외국인 지분율 변화 |
| 환율 민감도 | `exchange beta` | 환율 수익률에 대한 종목 수익률 민감도 |
| 동조성 | `mean_corr` | Lift 기준 파트너 산업과의 월별 평균 상관계수 |

### 처리 흐름

1. 종목별 일별 수익률과 환율 수익률을 정렬
2. 환율 베타와 외국인 수급 변화율 계산
3. 환율 민감도와 수급 변수를 표준화
4. K-Means로 종목 군집화
5. 군집 결과를 10개 산업으로 매핑
6. 산업별 상승/하락 이진 행렬 생성
7. Lift가 1.4 이상인 산업 쌍을 파트너 산업으로 정의
8. 월별 `mean_corr` 생성

## 성능

`mean_corr`를 추가했을 때 일부 산업에서 예측 성능 개선이 확인되었습니다.

| 산업 | Baseline RMSE | With `mean_corr` | 변화율 |
|---|---:|---:|---:|
| 자동차 | 0.00227 | 0.00187 | -17.6% |
| 유통·소매 | 0.00258 | 0.00244 | -5.4% |
| 에너지·정유 | 0.00275 | 0.00264 | -4.0% |
| 바이오·제약 | 0.00270 | 0.00268 | -0.7% |

자동차 산업에서는 RMSE가 약 17.6% 감소했고, 방향 적중률도 0.333에서 0.500으로 개선되었습니다.  
이는 산업 간 동조지수가 거시 변수만으로 설명하기 어려운 공통 요인을 보완할 수 있음을 보여줍니다.
